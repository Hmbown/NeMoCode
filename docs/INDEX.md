## How to Use This Documentation

1. **Find a topic**: Use this index or browse section READMEs
2. **Get canonical URLs**: Each doc lists official NVIDIA documentation URLs
3. **Check API details**: The `api/` section covers authentication and endpoints
4. **Find models**: The `models/` section catalogs all available model families
5. **Auto-update**: Run `docs/_scripts/sync-docs.sh` to refresh content from NVIDIA sources

## Documentation Freshness

These docs are structured references pointing to canonical NVIDIA sources. For the most current information, always follow the canonical URLs. Run the sync script periodically to check for new products or changed URLs.
