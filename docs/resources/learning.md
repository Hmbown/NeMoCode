# NVIDIA Learning Resources

NVIDIA provides free and paid self-paced courses through the Deep Learning Institute (DLI):

### Generative AI
- Building RAG Agents with LLMs

- Getting Started with AI on Jetson Nano

### NeMo Framework
- Building Custom LLMs with NeMo
- Fine-Tuning LLMs with NeMo Framework
- Data Curation with NeMo Curator

### Inference
- Deploying Models with Triton Inference Server
- Optimizing Inference with TensorRT

### Speech AI
- Building Speech AI Applications
- ASR with NeMo
