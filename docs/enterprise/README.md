# NVIDIA AI Enterprise

> Enterprise AI software platform with production support and lifecycle management.

## Canonical URL

`https://docs.nvidia.com/ai-enterprise/index.html`

## Overview

NVIDIA AI Enterprise is the software platform that packages NIM, NeMo, and other NVIDIA AI technologies with enterprise-grade support, security, and lifecycle management. It's the commercial offering for production deployments.

## What's Included

| Category | Components |
|---|---|
| **Inference** | NIM Microservices, Triton, TensorRT |
| **Training** | NeMo Framework, Megatron-Core |
| **Orchestration** | Run:ai, GPU Operator, NIM Operator, Network Operator |
| **Platform** | Container Toolkit, GPU Drivers, CUDA |
| **Applications** | Omniverse, pre-trained models |
